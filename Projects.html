My Projects

Netflix Clone

The Netflix Clone is a feature-rich web application designed to replicate the functionality and interface of the popular streaming service.
This project involved creating a visually appealing and user-friendly layout that mirrors the real Netflix experience.
Users can browse through a catalog of movies and TV shows, view trailers, and get information about each title.
The application is powered by the TMDB API, providing dynamic content updates based on real-time data.
Implementing Firebase for backend operations enabled secure user authentication and seamless integration of personalized features like saving movies to a watchlist.
This project honed my skills in API integration, responsive design, and frontend development.

Key Features

  • Real-time data fetching using TMDB API
  • Secure user authentication with Firebase
  • Mobile-friendly, responsive design

Technologies Used: React, Firebase, TMDB API

Amazon Clone

This E-commerce Platform is a comprehensive online shopping solution designed to simulate the end-to-end buying process.
The application includes essential features such as user registration, product search, filtering, and a fully functional shopping cart. A special focus was placed on building a secure checkout system and intuitive user interface.
The backend, powered by Node.js and MongoDB, ensures reliable data storage and server-side functionality.
Developing this platform allowed me to delve into database management, user authentication, and REST API development.
Additionally, I emphasized creating a responsive and accessible experience for a wide range of devices

Key Features

  • Secure user registration and login
  • Dynamic product catalog with search and filtering
  • Responsive shopping cart with real-time updates

Technologies Used: React, Node.js

Burger Restaurant Website

This Food Ordering Website offers a streamlined experience for browsing restaurant menus, customizing orders, and tracking delivery statuses.
The project began with designing an intuitive interface that simplifies the ordering process.
A key challenge was optimizing the site for various screen sizes to provide a consistent experience across devices.
JavaScript was utilized to handle order customization, including real-time price updates based on user selections.
This project not only strengthened my understanding of frontend development but also deepened my appreciation for user-centric design.

Key Features

  • Clean, intuitive design for menu navigation
  • Fully responsive layout for optimal display across devices
  • Well-organized content for easy readability and user experience

Technologies Used: HTML, CSS
